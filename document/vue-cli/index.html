# 安装

  1. 全局安装包：yarn global add @vue/cli，慢的话配置淘宝镜像。
  2. 安装完成后检查版本号：vue --version
  3. 如果版本较低升级版本：yarn global upgrade --latest @vue/cli

# 使用vue-cli创建项目
