## 新增的对象规则
### 对象属性、方法简写
1. 属性简写：当属性值与属性名相同时，可以只写属性名 
```JavaScript {.line-numbers}
    let name = 'zs';
    let age = 18;
    let obj1 = {
        name,
        age
    }
    //等价于：
    // let obj1 = {
    //     name: name,
    //     age: age
    // }
```
2. 方法简写

`方法名(参数) = {函数体}`
```JavaScript {.line-numbers}
    let obj2 = {
        fn(test) {
            console.log(`this ${test} function is running`);
        }
    }
    obj2.fn('test'); 
```
3. 键名引用

用方括号语法使用在键名上，读取外部变量的值作为键名
```JavaScript {.line-numbers}
    let name = 'userName';
    let obj2 = {
        [name]: 'ww',
        userAge: 17
    };
    console.log(obj2.userName);     //ww
```
