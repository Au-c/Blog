# 拨打电话

1. 应用场景：

iPhone上的Safari(还有些webkit android手机浏览器)会自动对看起来像是电话号码的数字串（包括已经加入连字符或括号格式化过的）添加电话链接，点击之后会询问用户是否想要拨打该号码。如果你不希望开启这个自动识别，可以将它关闭

2. 方法： - <meta>中: <meta name="format-detection" content="telephone=no" /> - 对确实需要拨号功能的号码: <a href="tel:13800138000">13800138000</a>

补充

<meta> 中的 content 属性亦可以控制是否弹出邮箱: content="email=no" 将不识别邮箱
